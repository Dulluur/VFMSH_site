8 Класс - Математика

Дополнительные материалы

  • 1. Квадратные уравнения. Квадратное уравнение, приведенное квадратное уравнение. Выделение полного квадрата (квадрата двучлена).
  • 2. Формула корней квадратного уравнения. Вывод формулы корней квадратного уравнения. Вторая формула корней квадратного уравнения (с четным вторым коэффициентом).
  • 3. Решение текстовых задач с помощью квадратных уравнений. Составление математической модели текстовой задачи. Решение и исследование полученных решений на их соответствие реальным данным задачи.
  • 4. Биквадратные уравнения. Биквадратное уравнение, решение с помощью замены переменной.
  • 5. Неполные квадратные уравнения. Определение, решение путем разложения на множители, решение путем нахождения квадрата неизвестного.
  • 6. Теорема Виета. Связь между корнями квадратного уравнения и его коэффициентами. Формулы Виета. Теорема, обратная теореме Виета.
  • 7. Разложение квадратного трехчлена на множители. Доказательство формулы разложения квадратного трехчлена на множители.
  • 8. Представление квадратного трехчлена в виде произведения двух линейных множителей с целыми коэффициентами. Сокращение дробей путем разложения квадратного трехчлена на множители.
  • 9. Целые корни уравнения с целыми коэффициентами. Прием отыскания целого корня уравнения с целыми коэффициентами среди делителей свободного члена.
  • 10. Решение уравнений третьей и четвертой степени. Решение разложением на множители многочлена в левой части. Решение понижением степени уравнения с помощью подстановки.
  • 11. Системы уравнений. Решение систем уравнений: способом подстановки, способом сложения, графически. Решение задач с помощью систем уравнений
